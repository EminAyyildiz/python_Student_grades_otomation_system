Bu projede sistem kullanıcıya yapabileceği toplamda 5 farklı seçenek sunmaktadır. bunları sayam gerekirse: 1- Dosyaya eklenen tüm öğrencilerin isimleri ve notlarını görebilirsiniz. 2- Dosyaya yani listeye yeni öğrenci ve notlarını ekleyebilirsiniz. 3- Bu seçenek ile kullanıcı gşrmiş olduğu notların harf karşılığını yeni bir dosyaya isimlerle beraber bastırabilecektir. 4- Bu seçenek sayesinde kullanıcı istediği öürencinin notlarını görebilecektir. 5- seçenek sistemden çıkış seçeneğidir. Dolayısıyla kullanıcı output olarak öğrencileri harf notlarıyla beraber görecektir. Ben kodu yazarken hem eğlendim hem öğrendim. Umarım sizlere de yararlı olur.



# My main purpose in this project is to design a system that reads the data in the txt file to be entered into the system, or if there is no such file, enters the grades of the students and then converts them into letter grades in order to facilitate our teachers. In this project, the system offers a total of 5 different options to the user. If I need to count them: 1- You can see the names and grades of all students added to the file. 2- You can add new students and their grades to the file, that is, to the list. 3- With this option, the user will be able to print the letter equivalent of the notes they have entered into a new file together with the names. 4- Thanks to this option, the user will be able to see the grades of the student he/she wants. 5- option is the option to exit the system. The system is placed in the while loop and will continue to work as long as the user does not want to exit. In addition, I would like to point out that the file I defined as "filename" in the code I uploaded here is the file with the notes. The user can replace this file with the final file if he/she wishes. Therefore, the user will see the students with their letter grades as output. While I was writing the code, I had fun and learned. I hope it will be useful to you too.
